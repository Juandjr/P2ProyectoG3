# Proyecto U2 Programacion web para integracion

## para subir al git (para miembros colaboradores) se debe realizar los siguientes pasos:

git init
git add .
git commit -m " "
git branch -M main
git remote add origin https://github.com/Juandjr/P2ProyectoG3.git
git push -u origin main

## Para clonar el repositorio en su equipo:

git clone https://github.com/Juandjr/P2ProyectoG3.git
