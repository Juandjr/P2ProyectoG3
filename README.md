# Proyecto U2 Programacion web para integracion

## para subir al git (para miembros colaboradores) se debe realizar los siguientes pasos:

1. git init
2. git add .
3. git commit -m " "
4. git branch -M main
5. git remote add origin https://github.com/Juandjr/P2ProyectoG3.git
6. git push -u origin main

## Para clonar el repositorio en su equipo:

1. git clone https://github.com/Juandjr/P2ProyectoG3.git

## Subir los cambios unicamente:

1. git add .    o  git add (nombre del archivo que quieran subir)
2. git commit -m "(explicacion del cambio o actividad realizada corto)"
3. git push -u origin main
